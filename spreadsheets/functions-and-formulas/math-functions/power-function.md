# POWER


---

## Content

The `=POWER()` function is used to raise a number to a specified power.

The syntax is:

```plain-text
=POWER(base, exponent)
```

The `base` is the number to be raised to the `exponent` power.

> ðŸ’¡ If the `base` is a negative number, the `exponent` must be an integer. If the `base` is a positive number, the `exponent` can be a decimal number.
